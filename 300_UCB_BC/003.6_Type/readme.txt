We discuss Scala/Chisel Type:
Topics:
1. Setup: Select Scala Interpreter and Run
2. Type in Scala: All objects in Scala have a type. Type is the class name of object.
3. Scala Type and Chisel Type: 
a. Scala type: Integer, Long, Short, false, Boolean, False, True, etc.
b. Chisel Type: hardware type, such as, UInt, SInt, 4.W, false.B, Reg(Bool()), Bool(), etc.
4. Type Match Operator: Scala/Chisel Match Type.
5. Unapply Method: Every Scala/Chisel have apply and unapply (reverse the apply) method.
6. Partial Function: Function that provide a subset of possible data.
7. Type Safe Checking/Connection: Bool, UInt, SInt, Clock, Bundle, etc.
8. Type Generics: for Polymorphism
9.  Chisel Type Hierarchy: we can us Visualize() to view the Type Hierarchy.
10. Chisel Type Generics <T>: Define custom type in hardware design.
11. Exercise/Solution for MAC (Multiply Accumulator)
12. Exercise/Solution: Integrator
13. Create Custom Type <T>
14. Example: DspComplex()
15. Exercise/Solution: Sign-Magnitude Numbers
