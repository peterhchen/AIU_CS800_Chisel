We discuss Network (Crossbar, Ring Network, Torus Network) Design.

2. Network Design with Abstract Class, Inheritance (Case class), Generic Tem,plate <T>, and Factory Pattern. 
3. Crossbar: cross every Inputs into each host.
4. Ring Network: Single/Double linked list for message forward/reversed.
5. Torus Network: 2-Dimensional row/column network
6. Generic Parameter input
7. Define abstract class
8. Define case class for single/multiple inheritance
9. Define/Parsing: Generic template <T>, 
10. Factory Design for crossbar, ring network, and Torus Network.
